# 📝 Гайд по синтаксису `quiz.json`

## 1. Общие правила (Чтобы не упал JS)
* **Кавычки:** Всегда двойные `"`. Одинарные `'` вызовут ошибку.
* **Запятые:** Ставятся между элементами. **Важно:** после последнего элемента в списке (массиве или объекте) запятую ставить **нельзя**.
* **Регистр:** `topicId` и `id` в темах должны совпадать копейка в копейку.

---

## 2. Секция "config" и "modes"
Здесь настраиваются правила игры.

```json
"modes": {
  "exam": {
    "name": "Зачет (8+)",
    "description": "Текст с переменными: {lives} жизни, {totalTime/60} мин.",
    "lives": 2,            // 0 — бесконечно
    "timePerQuestion": 15, // 0 — без таймера на вопрос
    "totalTime": 120,      // 0 — без общего таймера (в секундах)
    "endMessage": "Текст при успехе"
  }
}
```

---

## 3. Секция "topics" (Темы)
Здесь два типа тем: **обычные** и **комбинированные**.

### Обычная тема:
```json
{ 
  "id": "interference", 
  "name": "Интерференция",
  "distractorsCount": 2 // Сколько НЕправильных ответов подмешать к правильному
}
```

### Комбинированная (Итоговая) тема:
```json
{ 
  "id": "final_test", 
  "name": "ИТОГОВЫЙ ТЕСТ", 
  "includeTopics": ["huygens", "interference"], // Из каких ID брать вопросы
  "limitQuestions": 5, // Сколько случайных вопросов выбрать из общей кучи
  "distractorsCount": 3 
}
```

---

## 4. Секция "questions" (Вопросы)
Самая объемная часть. Здесь мы вставляем контент.

### Шаблон вопроса с картинкой:
```json
{
  "topicId": "interference", // Должен совпадать с ID из секции topics
  "mode": "all",             // "all" или конкретный id режима (например "exam")
  "text": "Текст вопроса. Можно использовать спецсимволы типа Δ, φ, π.",
  
  "image": "img/interference_1.png", // ПУТЬ К КАРТИНКЕ (локальный или URL)
  // Если картинки нет, просто удали эту строку или оставь "image": ""
  
  "options": [
    "ПРАВИЛЬНЫЙ ОТВЕТ (всегда ставь под индексом 0)", 
    "Неправильный 1", 
    "Неправильный 2", 
    "Неправильный 3",
    "Неправильный 4"
  ],
  "correct": 0, // Индекс правильного ответа (если один)
  "explanation": "Объяснение, которое появится после ответа (то самое МВД или палка)."
}
```

### Вопрос с МУЛЬТИВЫБОРОМ (несколько правильных):
```json
{
  "topicId": "diffraction",
  "text": "Выберите ВСЕ верные утверждения:",
  "options": ["Верно А", "Верно Б", "Неверно В", "Неверно Г"],
  "correct": [0, 1], // МАССИВ индексов правильных ответов
  "explanation": "Тут объясняем, почему А и Б верны."
}
```

---

## 🖼 Как работать с картинками
1. **Локально:** Создай папку `img` рядом с `quiz.html`. Клади туда файлы. В JSON пиши: `"image": "img/my_pic.png"`.
2. **Из интернета:** Просто копируй URL картинки (правой кнопкой в браузере -> "Копировать адрес изображения"). В JSON пиши: `"image": "https://site.com/pic.jpg"`.
3. **Размер:** Старайся не брать гигантские картинки (больше 2000px), иначе загрузка будет долго висеть на экране "Ждем загрузки картинки".

## 🛠 Чек-лист перед сохранением:
1. Не забыл ли я закрыть массив `]` в конце `options`?
2. Не поставил ли я лишнюю запятую после последнего вопроса?
3. Совпадает ли `topicId` вопроса с существующим `id` темы?
4. У всех ли вопросов есть поле `explanation`? (Без него будет `undefined` в интерфейсе).
